# Project Explanation

This project demonstrates how AI tools can help generate marketing content for local businesses.

The chosen business example is Habibs Hair and Beauty, a salon brand known for professional grooming and styling services.

Using prompt engineering techniques, the following marketing assets were created:

- UGC ad scripts
- Social media hooks
- Video generation prompts
- Social captions

These assets can be used to create short-form video advertisements for platforms such as:

- Instagram Reels
- Facebook Ads
- YouTube Shorts

The purpose of this project is to showcase how AI can simplify the content creation process for businesses that want to improve their online presence.
