# AI UGC Content Marketing Project

## Project Overview
This project demonstrates how AI and prompt engineering can be used to create UGC-style advertisements for a local business.

The example business used in this project is **Habibs Hair and Beauty**, a well-known salon brand offering professional hair styling, grooming, and beauty services.

The goal of this project is to design AI prompts that generate marketing content such as:
- UGC ad scripts
- Social media hooks
- Video generation prompts
- Social captions

This project shows how AI tools can assist small businesses in producing high-quality marketing content quickly and efficiently.

## Problem Statement

Many local businesses struggle to create engaging social media advertisements due to limited time, resources, and marketing expertise.

This project demonstrates how AI-generated UGC-style ads can help solve that problem by creating relatable and authentic marketing content.

## Example Use Case

A salon like Habibs Hair and Beauty can use AI-generated content to:

- Create Instagram Reels
- Run Facebook Ads
- Produce YouTube Shorts
- Promote services such as keratin treatment or haircuts

This reduces the time required to produce marketing content while maintaining quality.
